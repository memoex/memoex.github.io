h5read

  • permute - Rearrange dimensions of N-D array
    • permute(A, [4, 3, 2, 1])
    • size change: [3, 28, 28, 1000] => [1000, 28, 28, 3]
  • squeeze - Remove singleton dimensions
    • size change: [1, 28, 28] => [28, 28]

One Hot

labels = [0; 1; 2; 1];
one_hot = full(ind2vec(labels'+1))'

[~, indices] = max(one_hot, [], 2);
labels = indices - 1

Max and min

  • largest column numbers in each row: [~, indices] = max(A, [], 2)
  • largest in a matrix: max(A(:))

Data type

class(1) => 'double'

Rouding (取整)

  • floor rounds a number to the next smaller integer. (朝负无穷取整)
    • 3.5 => 3, -3.5 => -4
  • ceil rounds a number to the next larger integer. (朝正无穷取整)
    • 3.5 => 4, -3.5 => -3
  • round rounds a number to the nearest decimal or integer. (四舍五入)
    • 3.5 => 4, -3.5 => -4
  • fix rounds a number toward zero. (取整, 朝零取整)
    • 3.5 => 3, -3.5 => -3
