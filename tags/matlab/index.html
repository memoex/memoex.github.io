Rouding (取整)



    

  • floor rounds a number to the next smaller integer. (朝负无穷取整)


      

    • 3.5 => 3, -3.5 => -4
      • 

    • 

  • ceil rounds a number to the next larger integer. (朝正无穷取整)


      

    • 3.5 => 4, -3.5 => -3
      • 

    • 

  • round rounds a number to the nearest decimal or integer. (四舍五入)


      

    • 3.5 => 4, -3.5 => -4
      • 

    • 

  • fix rounds a number toward zero. (取整, 朝零取整)


      

    • 3.5 => 3, -3.5 => -3
      • 

    •
