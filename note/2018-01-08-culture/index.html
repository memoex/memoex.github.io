龙之九子

龙有九个儿子，是跟谁生的？为什么「龙生九子，各不成龙」？ - 豆子的回答 - 知乎 https://www.zhihu.com/question/23759686/answer/41997389

  1. 囚牛 x 母龙
  2. 睚眦 x 狼
  3. 嘲风 x 凤凰
  4. 蒲牢 x 蛤蟆
  5. 狻猊（suān ní） x 狮子
  6. 霸下、赑屃（bì xì） x 王八
  7. 狴犴（bì àn） x 老虎
  8. 负屃（fù xì） x 王八
  9. 螭吻、鸱吻、鸱尾（chī） x 鱼
