      复制图层
      command + J
       
    

    

      新建图层
      command + alt + shift + N
       
    

    

      solo图层
      alt + 鼠标左键单击图层
       
    

    

      撤销
      command + Z
       
    

    

      连续撤销
      command + alt + Z

      alt为复制，复制前必然要先选中

      shift + F5
       
    

    

      仿制图章
